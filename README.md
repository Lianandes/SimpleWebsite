# Simple Portfolio Website

Projeto desenvolvido para praticar a criação de um website de portfólio utilizando HTML, CSS e JavaScript.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em desenvolvimento front-end, responsividade e construção de interfaces modernas para apresentação pessoal e profissional.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Layout responsivo
- Interface moderna
- Seções organizadas
- Navegação simples
- Estrutura de portfólio pessoal

## 📚 Conceitos praticados

- Responsividade
- Estruturação de páginas
- Flexbox
- Manipulação visual com CSS
- Organização de componentes
